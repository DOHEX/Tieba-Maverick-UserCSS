# Tieba-Maverick-UserCSS

### 使用方法
1. 安装 **Stylus 扩展**：[Chrome](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne), [Firefox](https://addons.mozilla.org/fr/firefox/addon/styl-us/)
